# subpixelFix
Small vanilla js script that fixes blurry half pixel CSS transforms

## How to use
[Codepen](https://codepen.io/engray/pen/QQZZre)

![Subpixel problem explained](https://i.imgur.com/xxlTAK2.png)

Just call ```subpixelFix();```
